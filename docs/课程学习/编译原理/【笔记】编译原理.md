# 编译原理

**编译器**：对语言进行转化的程序。其源语言可以是 C，JAVA,GO 等，而目标语言通常是某种处理器的 ISA。

编译器与解释器不同。编译器的输出是可以在机器上执行的一个程序，而解释器则是直接根据输入，给出执行之后的结果。

IR：编译器使用一些数据结构表示它处理的代码，这些数据结构就被称为 IR。
